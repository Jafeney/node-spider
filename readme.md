# 基于node的网络爬虫项目
> @author Jafeney
> @dateTime 2016-08-05

## 项目说明
本项目采用`express`快速构建`HTTP`服务，用`Request`模块请求需要扒取的网页，然后用 `Cheerio`进行处理，然后结果以`JSON`格式存储。
```
    "body-parser": "~1.15.1",
    "cheerio": "^0.20.0",
    "cookie-parser": "~1.4.3",
    "debug": "~2.2.0",
    "ejs": "~2.4.1",
    "express": "~4.13.4",
    "morgan": "~1.7.0",
    "request": "^2.74.0",
    "serve-favicon": "~2.3.0"
```
